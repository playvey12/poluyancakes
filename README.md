# Kenwood Ambassador — партнерский лендинг (Angular)

Лендинг для амбассадора кухонных машин и насадок Kenwood. Сделан на Angular.

## Что делает сайт

- Рассказывает об амбассадоре
- Показывает каталог товаров
- При переходе открывается официальный магазин Kenwood
- **Автоматически применяется промокод амбассадора**

## Стек

- Angular
- TypeScript
- HTML5, CSS3

## Особенности

- Компонентная архитектура
- Адаптивная верстка
- Карточки товаров с динамической подстановкой промокода
